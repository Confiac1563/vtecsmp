Instructions to join the server:

  1. Click the button to get IP.
  2. Download and install Tailscale.
  3. Follow the instructions of Tailscale to log in.
  4. Return to Minecraft, go to Multiplayer and add the server with IP.
  5. Join and have fun!

    6. Notes:

  6. This server is running on Minecraft version 1.21.8
  7. Make sure to allow Tailscale through your firewall if prompted.
  8. If you encounter any issues, feel free to reach out via the Social page.
